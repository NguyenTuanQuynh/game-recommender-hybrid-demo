# Hybrid Game Recommender Demo

This project is a hybrid game recommendation web demo built from Amazon Video_Games data.

## Features
- Personalized homepage recommendation
- Similar games recommendation
- Search by title, description, and categories
- Event logging from user interactions
- Placeholder game cards without real posters

## Tech Stack
- Frontend: React + Vite
- Backend: FastAPI
- Data Processing: Python, Pandas, Scikit-learn

## Project Structure
- `backend/`: API and recommendation logic
- `frontend/`: user interface
- `scripts/`: preprocessing pipeline
- `data/`: raw and processed datasets
- `notebooks/`: EDA and experiments

## Dataset
Amazon Video_Games
